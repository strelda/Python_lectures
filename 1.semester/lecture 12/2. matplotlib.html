Matplotlib

import matplotlib.pyplot as plt
import numpy as np

Lines

Matplotlib by default works with data in the form of lists x = [1, 2, 3], or numpy arrays x = np.array([1,2,3]).

In[2]:
x = np.linspace(0, 10, 100)
y = np.sin(x)

plt.plot(x, y)
plt.xlabel('x')
plt.ylabel('sin(x)')
plt.title('matplotlib.pyplot')
plt.show()

First plt.plot() creates a canvas and following plots are drawn on it, until plt.show() or plt.savefig() is called.

In[3]:
x = np.linspace(0, 10, 30)
y1 = np.sin(x)
y2 = np.tan(x)
y3 = np.cos(x)

plt.plot(x, y1, 'bo-', label='sin(x)') # blue b, circle o, solid line -
plt.plot(x, y2, 'r--', label='tan(x)') # red r, dashed line --
plt.plot(x, y3, 'g^' , label='cos(x)') # green g, triangle up ^
plt.legend()
plt.ylim(-3,3)
plt.xlabel('x')
plt.ylabel('f(x)')
plt.savefig('sin_tan.pdf')

Scatter

https://matplotlib.org/stable/gallery/shapes_and_collections/scatter.html#scatter-plot https://matplotlib.org/stable/api/_as_gen/matplotlib.pyplot.scatter.html#matplotlib.pyplot.scatter

In[5]:
x_p = np.linspace(0, 10, 50)
y_p = np.random.random(50)

plt.scatter(x_p, y_p, marker='x', c=[[0,0,i/10] for i in x_p])
plt.hlines(0.5, xmin=0, xmax=10, linestyles='dashed') # draw a horizontal line
plt.hlines([0.25,0.75], 0, 10, colors='green') # y coordinate can be added as a list, xmin and xmax can be used as positional arguments
plt.annotate('zde', (6, 0.1), xytext=(7, 0.2), arrowprops=dict(facecolor='black', shrink=0.05), fontsize=20) # put a text with an arrow on a coordinate
plt.text(2, 0.1, 'zde taky', fontsize=20) # put only text on a coordinate
plt.show()

Publication quality plots

If you work with LaTeX, it is best to match the font and font-size to the one used in your document. Also you should match the figure size to the file and not change it thereafter, otherwise the fonts will be scaled and look different.

In[4]:
plt.rcParams.update({
    "text.usetex": True,
    "font.family": "serif",
    "font.size": 10,
    "axes.labelsize": 10,
    "axes.titlesize": 10,
    "legend.fontsize": 10,
    "xtick.labelsize": 10,
    "ytick.labelsize": 10,
})
